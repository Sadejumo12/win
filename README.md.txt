Version Control:

This is a class of systems responsible for managing changes to computer programs,documents,large website,
or other collections of information.

GIT:
Git is a version control system used for tracking changes in computer files.It is generally used for
source code management in software development.

Process Of Commiting A Project To Github Repository:
- Create a New repository for existing project 
-Drop your project inside your VS Code
-Click on clone to select the created new repository
-Select terminal
-add your files to the repository by entering the command "git add filename" to add all your files
use the command "git add ."
-Then you have to commit your changes so git can track them with the command "git commit -m "adding files"
-Now you push to your git repository hosted with github by telling git to add remote location,
with this command:git remote add origin https://github.com/username/your-repo-name.git.
